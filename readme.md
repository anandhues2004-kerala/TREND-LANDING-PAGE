

# 🌐 Trend Landing Page (2026)

A modern and responsive landing page built using **HTML + Tailwind CSS**, designed to showcase trending fashion or products for the year 2026.

---

## 📌 Project Overview

This project is a stylish landing page featuring:

* A responsive navigation bar
* Hero section with visuals and text
* Call-to-action buttons
* Social media icons
* Clean and modern UI using Tailwind CSS

Perfect for beginners learning **frontend development** and **Tailwind CSS layouts**.

---

## ✨ Features

* ⚡ Fully responsive design (Mobile → Desktop)
* 🎨 Modern UI with custom fonts
* 🧭 Navigation bar with menu items
* 🔍 Search button & icons
* 🖼️ Image-based hero section
* 📩 Call-to-action (Join Now)
* 🌐 Social media section

---

## 🛠️ Tech Stack

* **HTML5**
* **Tailwind CSS (CDN)**
* **Font Awesome Icons**
* **Google Fonts**
